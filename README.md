# Woolies X API coding exercise

### Base URL 
https://wooliesx.herokuapp.com/api/answers

### Framework

- Node 10.x
- JavaScript ES6 language features
- Express 4.x

### Plugins

- Babel 7.x

### Code quality control

- eslint
- prettier
- unit tests

### Installation

```
git clone git@github.com:ChristianRich/wooliesx.git
cd ./wooliesx
yarn build
yarn start
```

# Tests

```
yarn test
```
